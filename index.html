Project Desciption

This project focuses on developing an automated lock system for a pool table integrated with a time management system to enhance efficiency and fair usage. A prototype was built using a custom wooden pool table equipped with six pocket holes controlled by servo motors. These motors automatically lock the pockets when the session time expires, preventing further play until additional time is purchased. An Arduino microcontroller acts as the bridge between hardware and software, communicating with a Windows Forms application that manages session time, pricing, and user data. The software tracks playtime in real time, updates pricing based on selected durations, and stores session history in a Firebase database. The Arduino IDE is used to program the servo motors, ensuring that they function automatically based on time constraints.

The system consists of three main components: the software interface, hardware integration, and data management. The Windows Forms application provides an intuitive interface where administrators can monitor and manage table usage. Each tableâ€™s playtime and cost are displayed using a DataGridView, and admin staff can extend sessions with predefined time increments. A history feature allows for reviewing past sessions and generating earnings reports. The hardware integration includes servo motors installed in the pocket holes, which are triggered via the Windows Forms application through the Arduino microcontroller. When a session ends, the application signals the Arduino, which then activates the servo motors to lock the pockets automatically. For data management, Firebase Cloud stores session records, including table numbers, playtime, pricing, and timestamps, enabling real-time monitoring and administrative control. The system, developed using C#, eliminates manual time tracking, enforces fair play, and streamlines the management of pool tables through automation.
